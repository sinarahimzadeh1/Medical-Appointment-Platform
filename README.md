###

<p align="left">An advanced online medical appointment system with SMS authentication, role-based access control, smart doctor suggestions, appointment booking, blogging, rating system, and a professional contact module with map-based location support.</p>

###

<h2 align="left">About Project</h2>

###

<p align="left">A comprehensive online medical appointment system built with Vanilla PHP, featuring secure SMS-based authentication, a professional admin panel, and role-based access control.<br><br>The platform includes a smart doctor recommendation engine, advanced appointment booking with detailed visit descriptions, and a built-in blog system with post publishing, rating, and user feedback.<br><br>It also provides a professional contact module with real-time map-based location display, delivering a modern and intelligent healthcare experience for both patients and administrators.</p>
